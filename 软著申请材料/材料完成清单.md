### 步骤 8：提交申请

**方式一：线上申请（推荐）**

```bash
# 1. 登录官网
# https://www.ccopyright.com.cn/

# 2. 进入"版权登记"系统

# 3. 选择"计算机软件著作权登记"

# 4. 在线填报并上传材料扫描件

# 5. 缴纳费用
# - 普通办理：300 元
# - 加急办理：600-1500 元

# 6. 等待审核
# - 普通：30-35 工作日
# - 加急：3-10 工作日
```

**方式二：线下申请**

```bash
# 1. 准备纸质材料（一式一份）

# 2. 提交到受理大厅
# 地址：北京市西城区天桥南大街 1 号
#       天桥艺术大厦 A 座 302

# 3. 领取缴费通知单

# 4. 缴纳费用

# 5. 等待审核

# 6. 领取证书（自取或邮寄）
```
