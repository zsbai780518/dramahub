## 📝 填写说明

### 软件全称命名规则

```
[品牌/企业字号] + [功能用途] + [软件/系统/平台]

示例：
- DramaHub 海外短视频短剧发行平台软件
- 短剧海外发行管理系统软件
- 跨境短视频付费平台软件
```

### 版本号规则

```
V[主版本号].[次版本号].[修订号]

示例：V1.0、V1.0.1、V2.0
```

### 开发完成日期

- 填写实际开发完成的日期
- 不能晚于申请日期
- 建议填写为最近日期

### 软件分类

根据《计算机软件分类代码表》选择：
- **30100** - 基础软件
- **30200** - 工业软件
- **30300** - 应用软件 ← 选择此项
- **30400** - 嵌入式软件

## 🎯 注意事项

### 源代码文档要求

1. **页数要求：** 提交前后各 30 页，共 60 页
   - 不足 60 页的，提交全部代码
   - 每页不少于 50 行（结束页可少于 50 行）

2. **格式要求：**
   - A4 纸打印
   - 页眉标注软件名称和版本号
   - 页脚标注页码
   - 不得有空行和空白页

3. **内容要求：**
   - 包含核心功能模块代码
   - 不得出现其他公司版权信息
   - 代码应连续、完整

### 用户手册要求

1. **页数要求：** ≥15 页
2. **内容要求：**
   - 软件界面截图
   - 功能操作说明
   - 系统配置说明
3. **格式要求：**
   - A4 纸打印
   - 图文并茂
   - 清晰易懂

### 其他注意事项

1. **所有文档不得涂改**
2. **签名处需手写签名**
3. **复印件需加盖公章（公司申请）**
4. **在线填写申请表后打印**

## 📤 提交流程

### 线上申请（推荐）

1. 访问 **中国版权保护中心官网**
   - 网址：https://www.ccopyright.com.cn/

2. **注册/登录账号**
   - 个人申请：实名认证
   - 公司申请：企业实名认证

3. **在线填写申请表**
   - 选择"计算机软件著作权登记"
   - 按要求填写软件信息
   - 上传申请材料扫描件

4. **缴纳费用**
   - 官方费用：300 元/件（普通办理）
   - 加急费用：另计

5. **等待审核**
   - 普通办理：30-35 个工作日
   - 加急办理：3-10 个工作日

### 线下申请

1. **准备纸质材料**
   - 打印所有申请材料
   - 加盖公章（公司申请）
   - 装订成册

2. **提交到受理大厅**
   - 中国版权保护中心
   - 地址：北京市西城区天桥南大街 1 号天桥艺术大厦 A 座 302

3. **领取证书**
   - 审核通过后领取证书
   - 或选择邮寄送达

## 📞 常见问题

### Q1: 个人可以申请吗？

**答：** 可以。个人和公司都可以申请软件著作权。

### Q2: 申请需要多长时间？

**答：** 
- 普通办理：30-35 个工作日
- 加急办理：3-10 个工作日（额外付费）

### Q3: 软件著作权有效期多久？

**答：** 
- 个人：作者终生 + 死后 50 年
- 公司：首次发表后 50 年

### Q4: 未发表的软件可以申请吗？

**答：** 可以。开发完成的软件无论是否发表都可以申请。

### Q5: 源代码不足 60 页怎么办？

**答：** 提交全部源代码即可，无需凑够 60 页。

### Q6: 需要提交软件运行视频吗？

**答：** 不需要。只需提交源代码和用户手册即可。
