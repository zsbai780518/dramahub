## 1. 软件概述

### 1.1 软件简介

DramaHub 海外短视频短剧发行平台是一款面向全球市场的 B2B2C 短视频短剧发行管理系统。平台支持海外主流支付方式（PayPal、Stripe 等）和微信/支付宝跨境支付，为短剧内容方提供内容上传、审核、分发、付费、分润的全链路服务。

### 1.2 软件特点

- **全球化支付**：支持 PayPal、Stripe、微信支付国际版、支付宝国际收单
- **多端覆盖**：一套代码编译生成 iOS APP、Android APP、H5 网页
- **完整生态**：内容管理、订单支付、会员系统、分润结算
- **合规安全**：遵循 GDPR、CCPA 国际数据隐私法规

### 1.3 适用对象

- **C 端用户**：海外短剧观看用户
- **B 端内容方**：短剧出品公司、MCN 机构、内容创作者
- **平台运营方**：平台管理员、运营人员、客服人员

## 3. 安装部署

### 3.1 后端部署

#### 步骤 1：环境准备

```bash
# 安装 LAMP 环境
yum install httpd mysql-server php php-mysql php-redis -y

# 启动服务
systemctl start httpd
systemctl start mysqld
systemctl enable httpd
systemctl enable mysqld
```

#### 步骤 2：部署代码

```bash
# 上传代码到服务器
cd /var/www/html
git clone https://github.com/zsbai780518/dramahub.git backend
cd backend

# 设置权限
chmod -R 777 runtime/
chmod -R 777 public/uploads/
```

#### 步骤 3：配置数据库

```bash
# 创建数据库
mysql -u root -p

CREATE DATABASE dramahub DEFAULT CHARSET=utf8mb4;
exit

# 导入数据
mysql -u root -p dramahub < database.sql
```

#### 步骤 4：配置文件

编辑 `config/database.php`：

```php
return [
    'default' => 'mysql',
    'connections' => [
        'mysql' => [
            'hostname' => '127.0.0.1',
            'database' => 'dramahub',
            'username' => 'root',
            'password' => 'your_password',
            'hostport' => '3306',
        ],
    ],
];
```

#### 步骤 5：配置虚拟主机

编辑 `/etc/httpd/conf.d/dramahub.conf`：

```apache
<VirtualHost *:80>
    ServerName api.dramahub.com
    DocumentRoot "/var/www/html/dramahub/backend/public"
    
    <Directory "/var/www/html/dramahub/backend/public">
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

重启 Apache：

```bash
systemctl restart httpd
```

### 3.2 前端部署

#### 步骤 1：安装依赖

```bash
cd frontend
npm install
```

#### 步骤 2：配置 API 地址

编辑 `utils/request.js`：

```javascript
const BASE_URL = 'https://api.dramahub.com'
```

#### 步骤 3：编译打包

```bash
# H5 版本
npm run build:h5

# Android APP
npm run build:app-android

# iOS APP
npm run build:app-ios
```

## 5. 操作指南

### 5.1 C 端用户操作流程

#### 注册登录

1. 打开 APP 或 H5 页面
2. 点击"注册/登录"按钮
3. 选择注册方式（邮箱/手机/第三方）
4. 填写信息并完成验证
5. 登录成功

#### 观看短剧

1. 浏览首页或分类列表
2. 点击感兴趣的短剧封面
3. 进入详情页查看简介
4. 点击"立即观看"
5. 免费试看前 3 集

#### 付费解锁

1. 试看结束后弹出付费提示
2. 选择付费方式（单集/全集/会员）
3. 选择支付渠道（PayPal/微信/支付宝）
4. 完成支付
5. 解锁成功，继续观看

### 5.2 B 端内容方操作流程

#### 入驻申请

1. 访问内容方后台
2. 点击"入驻申请"
3. 填写公司信息、上传资质
4. 提交审核
5. 等待审核结果通知

#### 上传内容

1. 登录内容方后台
2. 点击"内容管理" → "上传短剧"
3. 填写剧名、简介、演员信息
4. 上传视频文件（支持批量）
5. 设置付费模式、价格
6. 提交审核

#### 查看收益

1. 登录内容方后台
2. 点击"数据统计" → "收益数据"
3. 查看总收益、分润明细
4. 点击"提现管理"
5. 填写提现金额、提交申请

### 5.3 平台运营操作流程

#### 内容审核

1. 登录运营后台
2. 点击"内容管理" → "待审核"
3. 查看提交的内容
4. AI 初审 + 人工复审
5. 审核通过/驳回

#### 订单管理

1. 登录运营后台
2. 点击"订单管理" → "订单列表"
3. 查询订单、查看详情
4. 处理异常订单
5. 审核退款申请

## 6. 常见问题

### Q1: 如何修改支付渠道配置？

**答：** 登录运营后台 → 系统管理 → 支付配置，选择对应支付渠道进行修改。

### Q2: 视频上传失败怎么办？

**答：** 
1. 检查视频格式（支持 MP4、MOV）
2. 检查文件大小（单集不超过 500MB）
3. 检查网络连接
4. 联系技术支持

### Q3: 分润比例如何设置？

**答：** 登录运营后台 → 内容方管理 → 分润设置，可设置默认分润比例或单独设置。

### Q4: 如何查看用户反馈？

**答：** 登录运营后台 → 运营管理 → 意见反馈，查看并回复用户反馈。

### Q5: 系统支持哪些语言？

**答：** 目前支持中文、英文、泰语、印尼语、西班牙语、阿拉伯语等 6 种语言。
