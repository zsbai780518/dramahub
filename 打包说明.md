## 📋 材料准备流程

### 第 1 步：填写申请人信息

**需要填写的文件：**
1. `申请表/申请信息表.md`
2. `证明材料/证明材料模板.md`

**填写内容：**
- [ ] 著作权人名称（公司/个人）
- [ ] 统一社会信用代码/身份证号
- [ ] 地址、联系人、电话
- [ ] 其他相关信息

### 第 2 步：准备源代码文档

**操作步骤：**

1. **打开源代码文件**
   ```
   源代码/后端源代码.txt
   源代码/前端源代码.txt
   ```

2. **复制到 Word 文档**
   - 新建 Word 文档
   - 设置字体：Consolas 或 Courier New
   - 设置字号：小四（10pt）
   - 设置行距：单倍行距

3. **添加页眉页码**
   - 页眉：`DramaHub 海外短视频短剧发行平台软件 V1.0`
   - 页脚：`第 X 页 / 共 60 页`

4. **确保页数**
   - 前后各 30 页，共 60 页
   - 每页不少于 50 行代码
   - 不足 60 页提交全部代码

5. **导出 PDF**
   - 文件 → 另存为 → PDF
   - 命名：`源代码文档.pdf`

### 第 3 步：准备用户手册

**操作步骤：**

1. **打开用户手册**
   ```
   用户手册/用户手册.md
   ```

2. **复制到 Word 文档**
   - 新建 Word 文档
   - 设置标题样式
   - 添加目录

3. **添加截图**
   - 登录界面截图
   - 首页截图
   - 播放界面截图
   - 支付界面截图
   - 个人中心截图
   - 管理后台截图

4. **确保内容完整**
   - 软件概述
   - 运行环境
   - 安装部署
   - 功能说明（带截图）
   - 操作指南
   - 常见问题

5. **确保页数**
   - ≥15 页

6. **导出 PDF**
   - 文件 → 另存为 → PDF
   - 命名：`用户手册.pdf`

### 第 4 步：填写申请表

**在线填写步骤：**

1. **访问官网**
   ```
   https://www.ccopyright.com.cn/
   ```

2. **注册/登录**
   - 个人申请：实名认证
   - 公司申请：企业实名认证

3. **选择业务**
   - 版权登记 → 计算机软件著作权登记

4. **在线填报**
   - 参考 `申请表/申请信息表.md`
   - 按步骤填写信息
   - 上传相关材料

5. **打印申请表**
   - 下载 PDF
   - A4 纸打印
   - 签名处手写签名
   - 公司申请加盖公章

### 第 5 步：准备身份证明材料

**公司申请：**
- [ ] 营业执照副本复印件（加盖公章）
- [ ] 法定代表人身份证复印件

**个人申请：**
- [ ] 身份证正反面复印件（本人签字）

### 第 6 步：准备其他证明材料（如适用）

- [ ] 权利归属证明（合作开发/委托开发）
- [ ] 授权委托书（委托代理）
- [ ] 版本升级证明（升级版本）

### 第 7 步：装订材料

**装订顺序：**
1. 计算机软件著作权登记申请表
2. 身份证明材料
3. 源代码文档（60 页）
4. 用户手册（≥15 页）
5. 其他证明材料

**装订要求：**
- A4 纸打印
- 左侧装订
- 不得活页装订
- 不得有涂改

### 第 8 步：提交申请

**方式一：线上申请（推荐）**

```
1. 登录官网：https://www.ccopyright.com.cn/
2. 进入"版权登记"系统
3. 选择"计算机软件著作权登记"
4. 在线填报并上传材料扫描件（PDF 格式）
5. 缴纳费用
6. 等待审核
```

**方式二：线下申请**

```
1. 准备纸质材料（一式一份）
2. 提交到受理大厅
   地址：北京市西城区天桥南大街 1 号
         天桥艺术大厦 A 座 302
3. 领取缴费通知单
4. 缴纳费用
5. 等待审核
6. 领取证书
```

## ⚠️ 注意事项

### 材料要求

1. **所有材料不得涂改**
2. **签名处必须手写签名**
3. **复印件需加盖公章（公司）或签字（个人）**
4. **源代码不得出现其他公司版权信息**
5. **页眉页码必须规范**

### 时间要求

1. **开发完成日期**不能晚于申请日期
2. **首次发表日期**如未发表填"未发表"
3. **申请提交后**材料不可修改

### 格式要求

1. **PDF 格式**：所有扫描件必须是 PDF 格式
2. **A4 纸张**：所有材料使用 A4 纸
3. **清晰可辨**：扫描件必须清晰
4. **文件大小**：单个文件不超过 10MB

## 🎉 常见问题

### Q1: 个人可以申请吗？

**答：** 可以。个人和公司都可以申请软件著作权。

### Q2: 申请需要多长时间？

**答：** 
- 普通办理：30-35 个工作日
- 加急办理：3-10 个工作日（额外付费）

### Q3: 软件著作权有效期多久？

**答：** 
- 个人：作者终生 + 死后 50 年
- 公司：首次发表后 50 年

### Q4: 未发表的软件可以申请吗？

**答：** 可以。开发完成的软件无论是否发表都可以申请。

### Q5: 源代码不足 60 页怎么办？

**答：** 提交全部源代码即可，无需凑够 60 页。

### Q6: 需要提交软件运行视频吗？

**答：** 不需要。只需提交源代码和用户手册即可。

### Q7: 可以加急办理吗？

**答：** 可以。加急办理 3-10 个工作日，费用 600-1500 元。

### Q8: 证书如何领取？

**答：** 可选择自取或邮寄。邮寄需到付。
